Privacy Policy for EasyShare

Last updated: June 2026

This Privacy Policy describes how EasyShare ("the Application") handles information when you use the Application.

The Application operates locally on the user's device and does not collect, store, or share personal information.

File Access

The Application accesses files and folders selected by the user solely to provide local file browsing and file sharing functionality. Files remain on the user's device and are not uploaded to any cloud service operated by the developer.

Network Usage

The Application may create a local HTTP server on the user's device to allow access to files from other devices on the same network, according to the user's actions and configuration.

License Verification

The Application uses Google Play Integrity API to verify that the Application has been obtained through Google Play and that the license is valid. Verification requests are processed through Google services. The developer does not collect or store personal account information during this process.

Analytics and Tracking

The Application does not use analytics services, advertising services, or tracking technologies.

Data Sharing

The developer does not sell, rent, or share user data with third parties.
